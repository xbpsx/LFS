Некоторые приложения могут настраивать своё поведение с помощью конфигурационных файлов, читаемых при запуске приложения. Часто приложения устанавливают версии конфигурационных файлов по умолчанию где-то под `/usr` или в каталоге `/etc`. Проверьте вывод команды `make install`, **чтобы увидеть, какие конфигурационные файлы были установлены.** Параметры конфигурации также должны быть задокументированы в README или INSTALL программы или на их веб-сайте.
