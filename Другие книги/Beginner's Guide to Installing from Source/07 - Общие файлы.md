Распакованные архивы исходного кода обычно содержат файл с именем README или INSTALL (или оба) в корневом каталоге. Вы всегда должны сначала прочитать эти документы, так как они дают важную информацию о том, как компилировать, устанавливать и настраивать остальной исходный код в загруженном архиве.

Одной важной частью информации, обычно находящейся в README или INSTALL, **является список другого программного обеспечения, которое должно быть установлено в первую очередь**. Любая программа, которую вы загружаете, потребует некоторых локальных заголовочных файлов, библиотечных файлов и/или вспомогательных инструментов для сборки или запуска. Если вы не установите предварительные требования правильно, то приложение может не скомпилироваться, или скомпилируется, но не запустится, или будет собрано без некоторых дополнительных функций, которые вам нужны.

Другая важная часть информации — **это набор параметров, которые могут быть переданы процессу сборки; см. далее.**
